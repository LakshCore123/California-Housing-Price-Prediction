# California Housing Price Prediction

## Project Overview
This project compares multiple machine learning regression models on the California Housing Dataset.

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Techniques
- Feature Scaling (StandardScaler)
- Train-Test Split
- RMSE Evaluation
- R² Score Comparison
- Model Performance Visualization

## Dataset
California Housing Dataset (Scikit-Learn)

## Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
